# React UI Sandbox Preview

Instantly preview React components from AI or the web in a secure sandbox, without starting a local development environment.

**Author:** Remco Stoeten

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)](https://vercel.com/stoeterscooter-9767s-projects/v0-component-evaluator-module)

## Overview

This is a React UI Sandbox Preview that allows developers to instantly preview and test React UI components in a secure environment without needing a local development setup.

## Deployment

Your project is live at:

**[https://vercel.com/stoeterscooter-9767s-projects/v0-component-evaluator-module](https://vercel.com/stoeterscooter-9767s-projects/v0-component-evaluator-module)**

## Features

- **Live Component Preview**: See your React components render in real-time
- **Code Editor**: Built-in code editor with syntax highlighting
- **File Management**: Create, edit, and organize component files
- **Secure Sandbox**: Components run in isolated environment
- **No Local Setup**: Start building immediately without installation

## Getting Started

### Prerequisites

This project is optimized for **Bun** but also supports **pnpm** and **npm**.

### Installation

#### With Bun (Recommended)
```bash
bun install
```

#### With pnpm
```bash
pnpm install
```

#### With npm
```bash
npm install
```

### Development Server

#### With Bun (Recommended)
```bash
bun run dev
```

#### With pnpm
```bash
pnpm run pnpm:dev
```

#### With npm
```bash
npm run npm:dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build and Deploy

### Production Build

#### With Bun (Recommended)
```bash
bun run build
```

#### With pnpm
```bash
pnpm run pnpm:build
```

#### With npm

```bash
npm run npm:build
```
